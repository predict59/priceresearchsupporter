# 변경 이력

이 문서는 GitHub에서 각 버전에 어떤 변경이 반영되었는지 빠르게 확인하기 위한 기록입니다.
새 기능을 배포할 때마다 최신 항목을 위에 추가합니다.

## 2026-07-02

### 매장 용어 정리 및 담당매장 기반 지도/리스트 개편
- 커밋: `6e43bd4`
- 화면에서 사용하던 `마트` 용어를 `매장` 중심으로 정리했습니다.
- 매장리스트 화면 안에 `매장 리스트`, `매장 지도`, `담당매장 설정` 탭을 추가했습니다.
- 담당매장으로 체크한 매장만 리스트와 지도에 표시되도록 변경했습니다.
- 담당매장 설정에서 매장을 한 줄 목록으로 빠르게 확인하고 전체 선택/해제할 수 있습니다.
- 지도 위치 검색을 `추가 위치`와 `전체 갱신`으로 나눴습니다.
- 좌표 검색 전 담당매장 수와 예상 처리 시간을 확인하도록 했습니다.
- 주소 검색 시 우편번호와 매대/코너 안내 문구를 줄이고, 도로명 주소의 번지 중심으로 검색하도록 보완했습니다.

### 무료 지도 1차 구현
- 커밋: `58cd1a2`, `진행 중`
- Leaflet + OpenStreetMap 배경지도를 사용해 무료 방향의 지도 화면을 추가했습니다.
- 마트리스트 필터 패널에서 `지도 보기`로 현재 지역의 지도 대상 마트를 확인할 수 있습니다.
- 마트 카드 메뉴에서 `지도 대상 포함/제외`를 설정할 수 있습니다.
- 지도 대상에서 제외한 마트는 지도 화면에 표시하지 않습니다.
- 엑셀 업로드 시 `위도/경도`, `lat/lng`, `x/y` 계열 컬럼이 있으면 마트 좌표로 저장합니다.
- 좌표가 없는 마트는 `좌표 검색` 버튼으로 주소 기반 좌표를 1초에 1건씩 검색해 저장합니다.
- 좌표 검색 시 주소 앞 우편번호는 제거합니다.
- 좌표가 있는 마트는 완료 여부에 따라 완료 회색, 미완료 빨강 마커로 표시합니다.
- 좌표가 없는 마트는 지도 아래 `좌표 없음` 목록에 표시합니다.
- 브라우저 위치 권한을 허용하면 내 위치를 지도에 표시할 수 있습니다.
- 데이터 구조 변경: `SurveyStore`에 `mapIncluded`, `latitude`, `longitude`, `geocodeStatus`, `geocodedAt` 필드를 추가했습니다.

앞으로 기능을 수정하고 커밋할 때는 이 파일에 다음 내용을 함께 기록합니다.

- 커밋 해시

- 데이터 구조 변경 여부
- 주의사항 또는 되돌릴 기준

## 주요 누적 변경

### 데이터 및 백업
- 조사표/업체 연락처 엑셀 업로드 후 지역, 마트, 물품 작업환경을 생성합니다.
- 입력 자료와 현장 조사 데이터는 서버 DB가 아니라 브라우저 IndexedDB에 저장합니다.
- 지역별 백업과 전체 백업을 지원하고, 사진 Blob도 백업/복원 대상에 포함합니다.

### 지역/마트/물품 흐름
- 권역 명칭을 지역 중심 흐름으로 정리했습니다.
- 지역 선택, 마트리스트, 마트정보, 물품리스트, 가격정보 화면 구조를 정리했습니다.
- 마트리스트와 물품리스트에서 검색, 필터, 임의 순서 편집을 지원합니다.
- 마트 방문 순서를 사용자가 직접 조정할 수 있습니다.

### 마트 상태 관리
- 마트 전경사진을 등록한 뒤 마트 상태를 미확인, 영업 중, 폐업, 임시휴업 중 하나로 설정합니다.
- 폐업/임시휴업 처리 시 하위 물품 데이터를 일괄 변경합니다.
- 영업 중이 아닌 마트에서는 가격정보 저장을 제한합니다.

### 사진 관리
- 마트 전경사진, 제품진열사진, 제품정보사진, POS/영수증사진을 IndexedDB에 저장합니다.
- 사진 업로드 시 리사이징/압축을 적용해 저장 용량 부담을 줄였습니다.
- 사진 미리보기를 제공합니다.
- 사진 ZIP 내보내기 시 물품코드 기준 파일명으로 생성합니다.

### 가격정보 입력
- 진열여부에 따라 실물확인, 상태, 사진자료, 가격 입력 흐름을 분기합니다.
- 정상가/할인가 입력 시 천 단위 콤마를 표시합니다.
- 기준가격 대비 조사가격 경고/정상 안내를 표시합니다.
- 적격/부적격 판단 기준을 일부 자동화했습니다.
- 할인기간, 구두확인, 행사여부, 기간구분 처리를 정리했습니다.

### 자동 인식
- 제품정보사진에서 바코드를 자동 인식해 조사표 바코드와 비교합니다.
- 제품진열사진에서 가격 후보를 OCR로 추출해 정상가/할인가 입력 후보 칩으로 표시합니다.

### UI/모바일
- 모바일 우선 UI로 상단바, 검색창, 하단 저장바, 카드형 리스트를 정리했습니다.
- 마트명/물품명 길이에 따른 말줄임과 뱃지 밀림을 개선했습니다.
- 사진누락, 완료, 적격/부적격 등 상태 표시를 조정했습니다.
- 로딩 화면과 자체 저장공간 모달을 추가했습니다.
