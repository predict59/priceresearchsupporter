# 가격조사 앱 빠른 사용 가이드

이 앱은 현장에서 매장별 가격, 사진, 특이사항을 바로 입력하고 지역별 엑셀/사진 ZIP/백업 파일을 내려받기 위한 도구입니다.

데이터는 서버가 아니라 이 기기의 브라우저 저장공간에 저장됩니다. 다른 기기에서 이어서 쓰려면 반드시 백업 파일을 내려받아 복원해 주세요.

## 전체 흐름

1. 조사표 엑셀과 매장 연락처 엑셀을 업로드합니다.
2. 메인에서 조사할 지역을 확인합니다.
3. 필요한 경우 `⋮ > 담당매장 관리`에서 내가 방문할 매장만 체크합니다.
4. `작업`을 눌러 매장리스트로 들어갑니다.
5. 매장정보에서 전경사진, 영업상태, 조사일을 입력합니다.
6. 물품리스트에서 품목을 골라 가격정보와 사진을 저장합니다.
7. 메인에서 지역별 엑셀, 사진 ZIP, 백업 파일을 내려받습니다.

## 화면별 핵심

### 자료 업로드

처음 1회만 조사표 엑셀과 매장 연락처 엑셀을 넣습니다. 업로드가 끝나면 지역, 매장, 물품 목록이 이 브라우저 안에 만들어집니다.

### 메인

지역별 전체 진행률을 봅니다. 담당매장을 따로 지정한 경우에는 전체 진행률 아래에 담당매장 기준 진행상황도 함께 표시됩니다.

메인에서 할 수 있는 일:

- `작업`: 해당 지역 매장리스트로 이동
- `엑셀`: 지역별 조사 결과 엑셀 다운로드
- `사진`: 지역별 사진 ZIP 다운로드
- `백업`: 지역별 백업 JSON 다운로드
- `⋮ > 담당매장 관리`: 내가 방문할 매장만 선택

### 담당매장 관리

기본값은 전체 미선택입니다. 내가 맡은 매장만 체크하면 매장리스트와 매장지도에는 체크한 매장만 표시됩니다.

위치정보가 필요한 경우:

- `없는 위치만 가져오기`: 체크한 매장 중 좌표가 없는 매장만 검색
- `전체 위치 다시 가져오기`: 체크한 매장 위치를 다시 검색

### 매장리스트 / 매장지도

매장리스트는 검색, 필터, 정렬로 방문 매장을 찾는 화면입니다. 품목명, 품목코드, 바코드도 검색됩니다.

매장지도는 내 위치와 매장 좌표를 기준으로 방문 위치를 보는 화면입니다. 완료 매장과 미완료 매장을 마커로 구분합니다.

### 매장정보

먼저 매장 전경사진을 넣고 영업상태를 설정합니다. 상태는 미확인, 영업 중, 폐점, 임시휴업 중 하나입니다.

폐점 또는 임시휴업으로 바꾸면 해당 매장의 물품 데이터가 일괄 반영됩니다. 다시 영업 중으로 바꾸면 물품 입력값을 다시 조사할 수 있도록 초기화합니다.

### 물품리스트

품목코드, 품명, 바코드, 기준가격, 조사가격, 적격/부적격을 빠르게 확인합니다. 저장된 사진이 있으면 작은 미리보기도 표시됩니다.

### 가격정보

품목별로 다음을 입력합니다.

- 진열여부, 규격일치, 바코드일치
- 제품진열사진, 제품정보사진, POS/영수증사진
- 정상가, 할인 여부, 행사 여부, 할인기간
- 비정상진열 여부와 비고

사진이 부족하면 저장 전에 알려주지만, 현장 상황상 사진 없이 저장할 수도 있습니다.

## 주요 기능

### 엑셀/사진 ZIP 내보내기

지역 단위로 결과 엑셀과 사진 ZIP을 생성합니다. 사진 ZIP은 품목코드 기준으로 저장됩니다.

- `품목코드.1.jpg`: 매장 전경사진
- `품목코드.2.jpg`: 제품진열사진
- `품목코드.3.jpg`: 제품정보사진
- `품목코드.4.jpg`: POS 또는 영수증사진

### 바코드 자동 비교

브라우저가 지원하면 제품정보사진에서 바코드를 읽고 조사표 바코드와 비교합니다. 다르면 바코드 불일치로 바로 확인할 수 있습니다.

### 가격 후보 자동 추출

제품진열사진에서 가격처럼 보이는 숫자를 읽어 정상가/할인가 후보로 보여줍니다. 후보 칩을 누르면 가격 입력칸에 바로 들어가고, 사용자가 언제든 수정할 수 있습니다.

### 적격/부적격 표시

저장된 조사값을 기준으로 적격/부적격을 자동 표시합니다. 저가판매, 미판매, 미진열, 바코드 미등록, 폐점, 임시휴업 등은 부적격으로 판단합니다.

### 담당매장 지정

담당매장 관리에서 내가 맡은 매장만 체크하면 매장리스트와 매장지도에 해당 매장만 표시됩니다. 전체 진행률은 유지하면서 담당매장 진행률도 같이 볼 수 있어, 내 몫만 따로 관리하기 좋습니다.

### 지도와 거리순

매장 위치정보와 내 위치를 사용해 거리순으로 정렬할 수 있습니다. 매장지도에서는 마커 색상으로 완료/미완료를 빠르게 확인하고, 마커를 누르면 담당자 정보와 입력 화면으로 바로 이동할 수 있습니다.

### 백업/복원

기기나 브라우저가 바뀌면 데이터가 자동 연동되지 않습니다. 전체 백업을 내려받아 다른 기기에서 복원하면 업로드 자료, 입력값, 사진까지 현재 작업환경을 그대로 옮길 수 있습니다.

## 현장 팁

- 출발 전 담당매장을 체크하고 위치정보를 가져오면 지도와 거리순 정렬을 쓰기 쉽습니다.
- 사진을 많이 찍는 날은 중간중간 지역별 백업을 내려받아 두는 것이 안전합니다.
- 앱이 이상하게 오래된 화면을 보이면 새로고침하거나 앱을 완전히 닫았다 다시 열어 주세요.
- 휴대폰을 바꾸거나 다른 브라우저로 열면 기존 데이터가 보이지 않습니다. 이때는 백업 파일로 복원해야 합니다.
